Milled Mold

Rhino File

The goal of this assignment is to create a mold for a plaster casting. The steps include
first creating a mold from wax, to be used to create a mold from silcon, to be used for
creating a object of desire in plaster.

I looked for inspiration on Thingaverse.Thinking about the movement of the milling
bit, I searched for ideas that didn't include an overhanging ledge. I chose a pyramid
because of it's top down unfolding nature.

I borrowed the native STL file created by __________from thingaverse and opened it in Rhino,
where I had created a 3d mesh container that matched the size of a block of I intended
to mill.

In Rhino, I placed the pyramid Stl file inside my mesh container,in contact with its base.
As a 'whole', I exported the file as a (Stl) file, one step toward manifesting the file
in G-Code, a language that the milling machine can interpret to carve my wax mold.


I opened my STL file in Fusion 360 where I was able to create and see or 'verify' a
milling tool path....But PIVOT. Nadia who was the makerspace saw my pyramid shape
and made the observaton that my pyramid had tiny detailing, to detailed for the
mill process to render.

I started over with looking for a simple iteration of a pyramid. I followed the
steps above to produce a milling pattern for the 'simple' pyramid.
